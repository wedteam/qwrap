String  >  s.tmpl 字符串模板

语法：

s.tmpl(opts)

参数：

类型描述
optsany(Optional) 模板中用到的变量opts。
如果没有此参数，s.tmpl()返回一个函数
如果有此参数，则相当于s.tmpl()(opts)，返回字符串

返回值：

如果没有opts参数，则返回模板函数。如果有opts参数，则返回模板套用变量后的结果

备注：

本方法由QW.StringH.tmpl经“methodize”变换得来

模板中变量以{$aaa}表示，相当于opts['aaa']。其它模板语法：
分隔符为{xxx}，"}"之前没有空格字符。
js表达式/js语句里的'}', 需使用' }'，即前面有空格字符
模板里的字符'{'用'##7b'表示
模板里的实体'}'用'##7d'表示
模板里的实体'#'可以用'##23'表示。例如（模板真的需要输出"##7d"，则需要这么写"##23#7d"）
{strip}...{/strip}里的所有\r\n打头的空白都会被清除掉
{}里只能使用表达式，不能使用语句，除非使用以下标签
{js ...}		－－任意js语句, 里面如果需要输出到模板，用print("aaa");
{if(...)}		－－if语句，写法为{if($a>1)},需要自带括号
{elseif(...)}	－－elseif语句，写法为{elseif($a>1)},需要自带括号
{else}			－－else语句，写法为{else}
{/if}			－－endif语句，写法为{/if}
{for(...)}		－－for语句，写法为{for(var i=0;i<1;i++)}，需要自带括号
{/for}			－－endfor语句，写法为{/for}
{while(...)}	－－while语句,写法为{while(i-->0)},需要自带括号
{/while}		－－endwhile语句, 写法为{/while}
