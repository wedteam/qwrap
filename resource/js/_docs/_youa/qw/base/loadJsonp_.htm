loadJsonp 静态方法 load一个JsonP地址。

语法：

loadJsonp(url,callback,options)

参数：

类型描述
urlstringJavascript文件路径
callbackfunction(Optional) json数据的调用函数
optionsObject配置选项，目前除支持loadJs对应的参数外，还支持：
{RegExp} callbackReplacer (Optional) 回调函数的匹配正则。
    默认是：/%callbackfun%/ig；如果url里没找到匹配，则会添加“callback=%callbackfun%”在url后面
{function} oncomplete (Optional) Javascript加载完成后还可以再执行一个函数。

返回值：

无
