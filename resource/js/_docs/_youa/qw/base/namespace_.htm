namespace 静态方法 获取一个命名空间，如果不存在，则创建并获取。

语法：

namespace(sSpace, root)

参数：

类型描述
sSpacestring命名空间符符串。如果命名空间不存在，则自动创建。
rootObject(Optional) 命名空间的起点。当没传root时：如果sSpace以“.”打头，则是默认为QW为根，否则默认为window。

返回值：

返回命名空间对应的对象，如果该命名空间不存在，则自动创建。
