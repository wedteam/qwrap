NodeW  >  w.insertTo 将元素插入到某个元素里

语法：

w.insertTo(sWhere, refEl)

参数：

类型描述
sWherestring 指定插入元素的地方
  1. beforebegin : 插入到元素前面
  2. afterbegin : 插入到元素内部的开头
  3. beforeend : 插入到元素内部的末尾
  4. afterend : 插入到元素后面
refElHTMLElement要插入到的元素

返回值：

w本身
