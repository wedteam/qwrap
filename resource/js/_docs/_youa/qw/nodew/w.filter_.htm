NodeW  >  w.filter 原型方法 在w的每个元素上运行一个函数，将函数返回真值的项拼成数组，并包装成NodeW返回。

语法1：

w.filter(callback, pThis)

参数：

类型描述
callbackfunction执行函数，传给函数的参数为(w[i],i,w)
pThisany(Optional) 指定callback的this对象

返回值：

在w的每个元素上运行一个函数，将函数返回真值的项拼成数组，并包装成NodeW返回。

语法2：

w.filter(selector)

参数：

类型描述
selectorstringcssSelector字符串。这里只支持最简selector，即里面没有关系符。

返回值：

用selector来过滤w的元素，将满足过滤的项拼成数组，并包装成NodeW返回。
