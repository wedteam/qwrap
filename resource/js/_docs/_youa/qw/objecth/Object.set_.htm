Object  >  Object.set 静态方法 设置对象属性

语法1：

Object.set(obj,prop,value)

参数：

类型描述
objobject对象
propstring属性名，深层属性用“.”区分表示，例如：style.color
valueany属性值

返回值：

w本身。

语法2：

Object.set(obj,func,args1)

参数：

类型描述
objobject对象
funcfunctionsetter函数
args1any(Optional) setter所需要的第二个参数。
argsnany(Optional) setter所需要的第(n+1)个参数。

返回值：

w本身。

语法3：

Object.set(obj,props)

参数：

类型描述
objobject对象
propsJSON属性名与属性值组成的JSON

返回值：

w本身
