Object  >  Object.mix 静态方法 将源对象的属性合入目标对象

语法1：

Object.mix(des,src,override)

参数：

类型描述
desobject合入目标对象
srcobject合入源对象
overrideboolean(Optional) 是否覆盖同名属性

返回值：

des

语法2：

Object.mix(des,srcs,override)

参数：

类型描述
desobject合入目标对象
srcsArray多个源对象组成的数组
overrideboolean(Optional) 是否覆盖同名属性

返回值：

des
