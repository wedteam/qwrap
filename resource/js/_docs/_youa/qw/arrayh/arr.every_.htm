Array  >  arr.every 在数组中的每个项上运行一个函数，若所有结果都返回真值，此方法亦返回真值。

语法：

arr.every(callback,pThis)

参数：

类型描述
callbackfunction运行函数，所接受的参数为(arr[i],i,arr)
pThisanycallback中this的指代变量

返回值：

boolean

本方法由QW.ArrayH.every经“methodize”变换得来

如果浏览器已支持every，则会调用浏览器默认的该方法
