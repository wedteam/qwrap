Function  >  Function.mul 将一个满足Helper规范的静态方法，变换成一个第一个参数支持数组的方法。

语法：

Function.mul(func, opt)

参数：

类型描述
funcfunction静态函数
optint(Optional) 特性参数。支持以下值：
1 表示getFirst将只操作第一个元素，
2 表示joinLists，如果第一个参数是数组，将操作的结果扁平化返回
else 如果第一个参数是数组，则将针对所有元素运行本方法，并返回所有结果组成的数组

返回值：

function 返回新的function
